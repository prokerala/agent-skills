# prokerala-astrology-api

Agent skill: developer reference for the [Prokerala Astrology API](https://api.prokerala.com) v2 (REST, any language). Covers all 112 operations with parameters, enums, and response schemas, plus guides for authentication, token caching, query parameter encoding, error handling, and common pitfalls.

The agent-facing entry point is [SKILL.md](SKILL.md); detailed documentation lives under [references/](references/).

## Install

```bash
npx skills add prokerala/agent-skills --skill prokerala-astrology-api
```

Or copy this directory into your agent's skills directory. See the [repository README](../README.md) for per-agent instructions and the companion [`prokerala-sdk-php`](../prokerala-sdk-php/) skill for the official PHP SDK.
