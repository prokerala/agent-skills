# Prediction

This section provides APIs that gives predictions based on the 12 zodiac signs. These include endpoints for Daily Prediction API, Advanced Daily Prediction API, and Daily Love Compatibility API, each offering insights for the given zodiac sign.

## Operations

| Method | Path | Summary | Details |
|--------|------|---------|---------|
| GET | `/horoscope/daily` | Daily Prediction | [View](../operations/get-horoscope-daily.md) |
| GET | `/horoscope/daily/advanced` | Advanced Daily Prediction | [View](../operations/get-horoscope-daily-advanced.md) |
| GET | `/horoscope/daily/love-compatibility` | Daily Love Prediction | [View](../operations/get-horoscope-daily-love-compatibility.md) |
